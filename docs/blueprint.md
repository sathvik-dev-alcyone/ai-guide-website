# **App Name**: REVA Issue Reporter Guide

## Core Features:

- Landing Page: A comprehensive landing page introducing the AI Maintenance Reporter.
- Header: Sticky navigation bar displaying the title and a wrench icon.
- Hero Section: Headline explaining how to report an issue quickly with a call-to-action button.
- Quick Flow: Visual guide illustrating the four-step process.
- Problem vs. Solution: Comparison of traditional reporting vs. the AI Maintenance Reporter.
- Step-by-Step Visual Guide: Detailed walkthrough of the reporting process with images and instructions.
- Issue Categorization: The tool will use uploaded images and text descriptions to automatically determine the maintenance issue type.

## Style Guidelines:

- Primary color: A vibrant blue (#29ABE2) to convey trust and efficiency.
- Background color: A very light blue (#F0F8FF) for a clean and calming feel.
- Accent color: A contrasting orange (#FF9933) for call-to-action buttons and highlights.
- Body and headline font: 'Inter' sans-serif for a modern and readable experience.
- Note: currently only Google Fonts are supported.
- Use simple and clear icons from a set like FontAwesome to represent different maintenance issues.
- Use a card-based layout to present each step of the reporting process clearly and concisely.
- Subtle animations when transitioning between sections to create a smooth user experience.
